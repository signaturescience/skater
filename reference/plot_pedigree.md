# Plot pedigree

Plot pedigree

## Usage

``` r
plot_pedigree(ped, file = NULL, width = 10, height = 8)
```

## Arguments

- ped:

  List of pedigree objects from
  [fam2ped](https://signaturescience.github.io/skater/reference/fam2ped.md)

- file:

  Output file path (must end in ".pdf")

- width:

  Width of output PDF

- height:

  Height of output PDF

## Value

No return value, called for side effects.
